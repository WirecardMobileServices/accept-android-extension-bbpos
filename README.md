## Overview
The library enables terminal extension for acceptSDK-Android solution. 

Supported devices:
  * [PosMate] (http://www.spirepayments.com/product/posmate/ "PosMate")
  * [SPm2] (http://www.spirepayments.com/product/spm2/ "SPm2")
